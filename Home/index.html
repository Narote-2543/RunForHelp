Run For Help :: หนึ่งคนวิ่งหลายชีวิตรอด

Even Date : 28 / 10 / 2566

มูลนิธิคุณธรรมการกุศลบัวใหญ่"เต็กก่า"จีหลักเกาะ จัดตั้งเมื่อ พ.ศ.2540 ปัจจุบันมี นางสาวกมลรัตน์ อภิเมธีธำรง เป็นประธานมูลนิธิฯ โดยมีวัตถุประสงค์เพื่อสาธารณประโยชน์ ช่วยเหลือเก็บศพไร้ญาติ เป็นหน่วยงานช่วยเหลืออุบัติเหตุจราจร บริจาคโลงศพ และช่วยเหลือภัยพิบัติต่างๆ ซึ่งในแต่ละปีจะมีการทำบุญทำทานแจกสิ่งของ จัดงานประเพณี ในเขตอำเภอบัวใกญ่และอำเภอใกล้เคียงจังหวัดนครราชสีมา แต่การดำเนินงานของมูลนิธิฯ ซึ่งมีงบประมาณไม่เพียงพอ จึงจัดทำโครงการเดิน-วิ่งเพื่อการกุศล Run For Help หนึ่งคนวิ่งหลายชีวิตรอด ครั้งที่1 เพื่อนำรายได้มาจัดซื้อรถอุปกรณ์กู้ภัย เพราะในปัจจุบันมีอุบัติเหตุทางรถยนต์เกิดขึ้นบ่อยครั้ง จึงมีความจำเป็นที่จะต้องจัดหาและจัดซื้ออุปกรณ์ช่วยชีวิตเพื่อช่วยเหลือชีวิตเเละบริการประชาชนในเขตรับผิดชอบได้อย่างครอบคลุม
